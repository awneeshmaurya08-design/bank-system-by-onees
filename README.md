# Banking System (Java)

This is a simple Java-based bank system that supports account creation, balance checking, withdrawal, Lockers and depositing.

## Features
- Withdraw money
- Deposit money
- Check balance
- PIN verification
- Apply for Lockers
- Apply for loans
